# Security policy

AuraOn is an experimental Arc testnet prediction-market project. Do not use it
with funds you cannot afford to lose.

Report suspected vulnerabilities privately to the repository owner through
GitHub Security Advisories. Do not include private keys, API keys, entity
secrets, user tokens, or recovery files in an issue, log, screenshot, or email.

High-impact reports include loss or locking of escrowed assets, unauthorized
market resolution, signer compromise, authentication bypass, and disclosure of
Circle or AI credentials. Please include affected commit/contract, reproduction
steps, and expected impact. Allow time for a fix and coordinated disclosure.

Only the current `main` branch is supported. A source-code fix does not modify
an already deployed immutable contract; the deployment address and version must
be checked separately.
